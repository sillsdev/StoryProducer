Créer une phase

Objectif : Dans la phase CRÉER, l'utilisateur doit compléter les crédits locaux page, prendre des décisions finales à propos des nouvelles vidéos qui seront créées et créer des vidéos pour les différents publics avec lesquels il/elle partagera les vidéos.

Activités:

  1. Complet la page Crédits locaux.  

  1. Appuyez sur le bouton MODIFIER LES CRÉDITS LOCAUX pour ouvrir une fenêtre contextuelle modifiable.

  2. Tapez les noms des personnes qui ont participé aux différentes activités de traduction, de composition de chansons, de photographie, de doublage, etc.

  3. Ajoutez un numéro de téléphone ou une adresse e-mail pour que les personnes puissent vous contacter si elles souhaitent obtenir plus de vidéos ou si elles ont des questions sur l'histoire.

  4. Appuyez sur ENREGISTRER pour fermer la fenêtre et revenir à la phase CRÉER.

  5. La coche grise dans le cercle deviendra verte lorsque la diapositive des crédits locaux aura été modifiée.

  1. Taper Le nom de l'histoire (dans votre langue) sur la ligne correspondante. Il apparaîtra dans le nom du fichier vidéo pour vous aider à identifier son contenu. (Le numéro de l'histoire sera automatiquement ajouté au nom du fichier vidéo.) Ce texte peut être modifié à tout moment. La coche grise dans le cercle deviendra verte une fois le texte saisi sur cette ligne.

  2. Choisir Options à inclure dans la vidéo. Appuyez sur la case correspondant à chaque option pour l'activer. Appuyez à nouveau pour la désactiver.

  3. Créer Votre nouvelle vidéo . Appuyez sur le bouton « CRÉER UNE VIDÉO » et attendez que l'application mobile génère les nouvelles vidéos. N'effectuez aucune autre action sur votre téléphone pendant la création. Cela prendra entre 1 et 30 minutes, selon la durée de l'histoire et de la chanson. Vous pouvez consulter la barre de progression pour suivre l'avancement. L'application créera automatiquement et simultanément votre histoire en deux formats vidéo : MP4 et 3GP.

  4. Recréer Votre vidéo . Vous pouvez créer plusieurs versions de votre story vidéo. Sélectionnez différentes options ou fonctionnalités pour créer plusieurs versions de votre story.

Remarques :

  • Le nom de fichier de chaque vidéo sera généré automatiquement par l'application : un numéro à 3 chiffres tiré du numéro du modèle d'histoire + un nom d'histoire courte dans la langue saisie par l'utilisateur dans la phase CRÉER + un code Ethnologue à 3 lettres pour la langue tirée des informations de langue d'enregistrement + divers codes courts à 2 lettres indiquant quelles options ou fonctionnalités sont dans la vidéo (Fx = musique de fond et effets ; Sg = chanson locale ; Px = images ; Mv = mouvement sur les images ; Tx = texte de l'histoire / sous-titres)

  • Le format 3gp est une vidéo basse résolution conçue pour être visionnée sur les petits écrans des téléphones classiques. Ces vidéos sont relativement petites et occupent peu d'espace de stockage (environ 20 Mo en moyenne par story), mais elles apparaîtront granuleuses ou pixelisées sur l'écran d'un smartphone.

  • Le format MP4 est une vidéo haute résolution conçue pour être visionnée sur des écrans plus grands et projetée. Ces vidéos sont plus volumineuses et nécessitent chacune en moyenne 200 Mo de stockage. Une vidéo haute résolution (MP4) est nécessaire pour lire les sous-titres et les transcriptions à l'écran.

  • Voici quelques exemples des types de médias que Story Producer peut créer :

    • Pour un film

      • Inclure les options pour les images et le mouvement des images (avec ou sans musique de fond).

    • Pour un livre parlé

      • Inclure les options pour les images et le texte de l'histoire, sans mouvement d'image (avec ou sans musique de fond).

      • Saisissez les légendes du texte de l'histoire dans la phase DRAME qui correspondent aux enregistrements audio.

    • Pour les pages fixes avec des légendes de texte (comme un livre, plus les changements de page automatiques)

      • Inclure les options pour les images et le texte de l'histoire.

      • Saisissez les légendes du texte de l'histoire dans la phase DRAME qui correspondent aux enregistrements audio.

      • Lisez-le à volume audio nul

    • Pour une narration audio avec musique de fond

      • Désactivez les images et le texte de l'histoire pour qu'il n'y ait pas de visuels

    • Tout ce qui précède avec ou sans chanson locale en annexe.

  • Options de référence

Inclure la musique de fond (Fx)

Inclure une chanson locale (Sg)

Inclure des images (Px)

Activer le mouvement de l'image (Mv)

Inclure le texte de l'histoire (Tx)
