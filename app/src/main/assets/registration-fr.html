Inscription

Général:

  1. L'inscription est encouragée lorsqu'une équipe locale commence à utiliser l'application Story Producer (SP) ou pendant la première partie d'un atelier de formation.

  2. Il est très utile de remplir tous les champs et de soumettre l'inscription pour que SIL Global puisse vous aider et poursuivre le développement de SP. (Si vous vous trouvez dans un endroit sensible, entrez un pseudonyme.)

  3. Appuyez sur chaque « tiroir » d'informations pour le développer et saisissez les informations. Faites défiler vers le bas pour accéder à chaque champ. Certains champs sont des menus déroulants sur lesquels vous devez appuyer pour choisir un élément. Appuyez à nouveau sur le « tiroir » pour le fermer.

  4. Envoyez les informations d'inscription par e-mail à SPapp_info@sil.org . Lors de l'envoi par e-mail, toutes les autres adresses e-mail répertoriées dans les champs d'inscription seront automatiquement insérées dans la section du destinataire de l'e-mail.

    • Si votre téléphone n'est pas connecté à Internet, les informations d'inscription (un petit document texte) peuvent être envoyées via Bluetooth au téléphone du formateur. Ensuite, lorsque le formateur se connecte, il peut soumettre à nouveau ou transmettre les informations d'inscription à SPapp_info@sil.org .

  1. Les informations d'inscription peuvent être modifiées et (re)soumises lorsque des informations mises à jour sont disponibles.

Informations linguistiques :

  • Le code Ethnologue est un code à 3 lettres qui a été attribué à chaque langue. Cliquez sur le lien QU'EST-CE QUE C'EST ? pour accéder à la base de données Ethnologue en ligne et rechercher un code spécifique. Si vous travaillez dans une langue ou un dialecte pour lequel aucun code n'a encore été attribué, veuillez saisir le code à 3 lettres de la langue la plus proche, suivi d'un tiret (sans espace) suivi immédiatement des 3 lettres principales du nom du dialecte. Par exemple, PEX-MAJ. Ce code, s'il est inséré et soumis, apparaîtra dans le titre partiellement automatisé de toutes les vidéos produites afin qu'il soit clair dans quelle langue la vidéo est produite.

  • Le statut de l'orthographe est lié à la nouveauté et à la stabilité du système d'écriture ou de l'alphabet de la langue. Appuyez sur le menu déroulant, puis sur l'option la plus appropriée.

Informations sur le traducteur :

  • Si vous souhaitez saisir des informations pour plusieurs traducteurs, insérez simplement une virgule ou une barre oblique et répertoriez les différentes personnes avec leurs informations correspondantes sur la même ligne.

Contacts supplémentaires :

  • C'est ici que d'autres adresses e-mail peuvent être saisies si une organisation ou une partie intéressée souhaite recevoir les informations d'inscription.

  • Si une organisation ou une partie intéressée souhaite recevoir des vidéos produites, un numéro WhatsApp ou une adresse e-mail peut être saisi ici pour permettre au chef de l'équipe SP locale de partager des vidéos avec eux.
