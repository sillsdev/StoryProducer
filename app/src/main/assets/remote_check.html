Remote Check

 

Who:  Usually one or two primary local language translator(s), perhaps with an Uninitiated Local Language Speaker who can answer comprehension questions posted by the ROCC, will monitor the text messages from the ROCC, make any necessary translation revisions, and communicate with the Accuracy Checker / ROCCer. . 

 

Purpose:  The purpose of the Remote Check phase is essentially a slide-by-slide instant messaging space or texting interface where the Accuracy Checker / ROCCer and the primary story translator(s) can communicate about the meaning of the translation, where suggestions for revisions can be encouraged, where questions can be asked and answered, where text back translations (transcriptions) can be requested and displayed, and where approval for each slide can be indicated.

 

Activities: 

  1. The translator(s) can respond to text messages from the ROCCer / Accuracy Checker.
  2. If the ROCCer requests a text back translation (transcription), this will need to be done in the Slide Tell Back phase.  When sent, the transcription will show up in a purple text box in the Remote Check phase.
  3. The translator can tap the triangular playback button under the image to listen to the most current translation draft for the slide. 
  4. If the translator(s) needs to make a revision to the translation draft, they must swipe up or use the dropdown menu to go to the Translate+Revise phase to orally record their new translation for the slide in question.  This revised draft will flow through to the Tell Back and Remote Check phases. 
  5. When the ROCCer is satisfied with the translation of a slide, s/he can approve that slide by sliding over the approval button for the slide.  When/If the SPadv phone is online, this slide approval will display as a green tick mark next to the picture. 

 

Notes:

●     The ROCCer should also check and approve the song slide (even if no song was recorded). 

●     When the ROCC has approved a slide, the gray tick mark will turn green for that slide.  The ROCCer controls all slide approvals.  Slide approvals can be reversed by the ROCCer and the tick mark will revert back to gray.  When all the slides are approved, and when the SPadv phone is online to receive the approvals, the Voice Studio and other final production phases will be unlocked.
