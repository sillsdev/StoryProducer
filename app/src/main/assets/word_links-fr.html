Liens vers des mots

À qui : Les traducteurs et les vérificateurs de précision trouveront cet outil ou cette fonctionnalité utile.

Objectif : L'objectif de l'outil Liens de mots est de fournir un accès pratique aux mots et noms difficiles dans toutes les histoires et de créer une base de données audio. Traduction en langue locale des mots et noms difficiles en un seul endroit. Aider les traducteurs à rendre les termes clés bibliques de manière cohérente et à prononcer les noms de personnes et de lieux de manière cohérente dans/à travers toutes les histoires. Fournir également des traductions littérales de texte pour tous les mots et noms difficiles (et les translittérations de noms) comme un autre moyen d'aider un vérificateur d'exactitude dans son travail.

Installation:

Pour installer une base de données WordLinks, copiez le fichier wordlinks().csv dans le dossier wordlinks du dossier SP Templates . Redémarrez ensuite l'application.

Activités:

  1. Dans le menu latéral, liste WordLinks :

    1. Tapez dans la barre de recherche pour trouver un terme ou un nom spécifique, ou faites défiler les mots classés par ordre alphabétique pour trouver ce que vous cherchez.

    2. Appuyez sur un mot/nom pour ouvrir la fenêtre contextuelle correspondant à ce terme.

  2. Appuyez sur un terme hypertexte dans la section texte de la phase TRADUCTION ou VÉRIFICATION DE L'EXACTITUDE de chaque diapositive.

    1. Un lien hypertexte bleu indique qu'il n'existe pas de rendu audio pour ce terme. En appuyant sur un lien hypertexte bleu, l'utilisateur sera redirigé vers l'écran de recherche du terme.

    2. Un lien hypertexte clair (texte normal souligné) indique qu'un rendu audio (au moins un) a été réalisé pour ce terme. En appuyant sur un lien hypertexte clair, l'utilisateur sera redirigé vers l'écran de documentation audio et textuelle du terme. Faites glisser votre doigt vers le haut pour voir l'écran de recherche.

  3. Lisez et recherchez la signification d'un terme. Discutez de la manière de traduire le terme dans le contexte de l'histoire dans la langue locale. Envisagez d'autres termes apparentés mais différents.

  4. Documentez un terme ou un mot en langue locale .

    1. Appuyez sur le bouton du micro en bas de l'écran pour enregistrer une traduction audio du terme dans la langue locale. Appuyez sur le bouton carré pour arrêter l'enregistrement.

    2. Appuyez dans la zone de texte associée pour saisir une traduction littérale mot à mot dans la langue source (LWC) du rendu audio du terme clé. Ou, si un nom a été enregistré, saisissez une translittération (phonétique) du mot.

  5. Rappelez-vous comment un mot a été rendu. Appuyez sur un lien hypertexte clair pour ouvrir la fenêtre contextuelle Lien vers le mot, écran de documentation. Appuyez sur le bouton de lecture en forme de triangle pour écouter comment un nom ou un terme a été rendu dans une autre histoire. Quittez la fenêtre contextuelle pour reprendre le travail de traduction.

  6. Quittez la fenêtre contextuelle Word Link en appuyant sur le X dans le coin supérieur droit de l'écran. Reprenez vos activités de traduction et de vérification. Parfois, appuyez sur la flèche de retour de votre téléphone pour revenir à l'endroit d'où vous venez.

Remarques :

  1. S’il existe une équipe de traduction des Écritures ou de traduction de la Bible en langue locale, les mêmes termes et noms clés doivent être utilisés afin qu’il y ait une cohérence dans la manière dont les termes et les noms sont rendus dans les histoires.

  2. Deux types de mots sont inclus dans la base de données WordLinks :

    1. Biblique clé termes . Certains de ces termes peuvent avoir plusieurs sens en fonction de leur contexte. Les traducteurs peuvent avoir besoin d'enregistrer plusieurs rendus audio pour un terme clé et de déterminer quel rendu utiliser dans quel contexte. Plusieurs rendus audio peuvent également devoir être enregistrés pour documenter différentes formes d'un terme (par exemple, la forme verbale et la forme nominale)

    2. Noms de personnes et de lieux. L'enregistrement d'un rendu audio des noms a pour objectif principal d'assurer une cohérence dans la prononciation de toutes les histoires.

  3. Il existe plusieurs sections dans l' écran de recherche de la fenêtre contextuelle Liens de mots pour aider le traducteur de l'histoire à décider comment restituer le terme :

    1. Le mot hyperlié de l'histoire suivi de ses différentes formes grammaticales. Celles-ci se trouveront en haut de la fenêtre contextuelle dans la section rouge.

    2. Notes – brèves définitions ou informations générales.

    3. Termes apparentés (mais différents), hyperliés – pour fournir du matériel de comparaison et de contraste, des recherches plus approfondies.

    4. Exemples de rendus d'autres langues, traduits en sens inverse - pour fournir des idées pratiques sur la manière dont les traducteurs d'histoires pourraient vouloir traduire le terme dans leur langue locale.

  4. La barre d'enregistrement en bas de l'écran s'étendra jusqu'à devenir un écran étendu sous l'écran de recherche une fois qu'un enregistrement aura été effectué pour un terme clé. Une fois les rendus audio enregistrés pour un terme, ils peuvent être consultés dans cet écran de documentation étendu : le rendu audio dans la langue locale peut être écouté et le texte rétrotraduit associé peut être lu.

  5. La base de données de langue locale WordLinks doit être sauvegardée régulièrement. Le dossier WordLinks se trouve à la racine du dossier de travail SP Templ ates qui peut être stocké sur le téléphone ou sur une carte SD. Pour la sauvegarde, copiez le répertoire WordLinks sur un autre lecteur ou une autre carte.

  6. L'auteur de toute base de données WordLinks (termes clés) est responsable de son contenu. Regardez à l'intérieur du fichier pour identifier son auteur.
